# restaurant-portfolio

店舗向け Web 制作のサンプル集です。現在は飲食店向け 1 枚 LP を 10 テーマで公開しています。

## 公開 URL

GitHub Pages 有効化後:

```
https://mech-logic-hub.github.io/restaurant-portfolio/
```

| # | サンプル | パス | 方向性 |
|---|----------|------|--------|
| 1 | 墨と余白 | `/sumi-breath/` | 墨・和紙・極少色・明朝 |
| 2 | 和モダン | `/wa-modern/` | 余白・直線・和紙トーン |
| 3 | 食べログ風 | `/tabelog-style/` | 白地・橙アクセント・情報カード型 |
| 4 | 温かみ | `/warm/` | クリーム・丸み UI |
| 5 | 町食堂 | `/machi-shokudo/` | 看板・チョークボード風 |
| 6 | 昭和レトロ | `/showa-retro/` | 提灯・暗木・赤・懐旧居酒屋 |
| 7 | ガーデン | `/garden-organic/` | 緑・丸フォント・ナチュラル |
| 8 | インスタ風 | `/instagram-soft/` | パステルグラデ・ストーリー風 |
| 9 | ポップコミック | `/pop-comic/` | 太枠・原色・擬音・漫画風 |
| 10 | ネオン夜 | `/neon-night/` | 黒地・ネオン・グロー |
| — | 一覧 | `/` | テーマ比較ページ |

テーマは「静けさ・抑制・伝統」から「装飾・情報密度・デジタル感」へと並んでいます（#1 が最も静か、#10 が最も装飾的）。

## 初回セットアップ（GitHub Pages）

1. リポジトリ **Settings → Pages**
2. **Source** を **GitHub Actions** に設定
3. `main` への push で自動デプロイ

## ローカルプレビュー

```bash
python3 -m http.server 8080
# http://localhost:8080/
```

## サンプル店舗（LP）

- 店名: **まちの灯食堂**（架空）
- コンセプト: ○○駅前の小さな食堂。昼は定食、夜は小皿
- 連絡先・住所はすべてサンプル用の架空データ

## ライセンス

制作サンプルとして自由に閲覧・参照できます。画像は Unsplash 経由のプレースホルダーです。

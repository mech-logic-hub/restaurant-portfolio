# restaurant-portfolio

店舗向け Web 制作のサンプル集です。現在は飲食店向け 1 枚 LP を 3 テーマで公開しています。

## 公開 URL

GitHub Pages 有効化後:

```
https://mech-logic-hub.github.io/restaurant-portfolio/
```

| サンプル | パス |
|----------|------|
| 一覧 | `/` |
| 温かみ | `/warm/` |
| 和モダン | `/wa-modern/` |
| 町食堂 | `/machi-shokudo/` |

## 初回セットアップ（GitHub Pages）

1. リポジトリ **Settings → Pages**
2. **Source** を **GitHub Actions** に設定
3. `main` への push で自動デプロイ

## ローカルプレビュー

```bash
python3 -m http.server 8080
# http://localhost:8080/
```

## サンプル店舗（LP）

- 店名: **まちの灯食堂**（架空）
- コンセプト: ○○駅前の小さな食堂。昼は定食、夜は小皿
- 連絡先・住所はすべてサンプル用の架空データ

## ライセンス

制作サンプルとして自由に閲覧・参照できます。画像は Unsplash 経由のプレースホルダーです。
